# 第五部分 沃土

项目与团队置身于更大的组织结构提供的环境中。我们称此环境为企业文化。一些文化支持健康地工作，另一些则不然。虽然组织层级因素可能会超出你能力控制的范畴，但它们仍然值得关注。最坏情况下，你需要认识什么是上层强加于你的；最好的情况下，你可能会在某一天能够身居其位，并对其进行改造，以利于下面的团队开展工作。
